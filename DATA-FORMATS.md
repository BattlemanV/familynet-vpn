# Data Formats

## Overview

FamilyNet VPN stores runtime data in JSON, SQLite and log files.
All runtime files live in `/data/` inside the container (`wg-vpn-data` volume).
Never commit these to Git.

---

## Runtime Files (inside container: /data/)

### api_token

Location: `/data/api_token` (read-only bind mount from `/root/wg-admin-api/api_token`)

Purpose: Recovery token for developer/SSH access.

Security: Secret. Never displayed on screen (only file path shown).

---

### clients.json

Location: `/data/clients.json`

Main client configuration file.

```json
{
  "server": {
    "privateKey": "...",
    "publicKey": "...",
    "address": "10.8.0.1"
  },
  "clients": {
    "u-u-i-d": {
      "name": "Admin",
      "address": "10.8.0.2",
      "privateKey": "...",
      "publicKey": "...",
      "preSharedKey": "...",
      "enabled": true,
      "role": "admin",
      "created_at": "...",
      "updated_at": "..."
    }
  }
}
```

Security: Contains private keys. Never commit.

---

### settings.json

Location: `/data/settings.json`

Panel settings.

```json
{
  "display_name": "Family VPN"
}
```

---

### speed_limits.json

Location: `/data/speed_limits.json`

Per-client speed limit settings.

```json
{
  "u-u-i-d": {
    "rate": "10mbit",
    "enabled": true,
    "updated_ts": 1718000000
  }
}
```

Rate format: `^\d+(kbit|mbit|gbit|kbps|mbps|gbps)$`

---

### activity.log

Location: `/data/activity.log`

JSON Lines. Each line is valid JSON.

```json
{"ts":1718000000,"event":"create","target":"peer_name","detail":"..."}
```

Known event types:
- `create`, `delete`, `enable`, `disable`
- `name`, `role`
- `speed-limit`, `speed-normal`
- `maintenance`, `restart-vpn`, `restart-admin`, `reboot-server`
- `backup`, `restore-backup`, `restore-backup-started`
- `parental-set`, `parental-delete`
- `avatar`
- `error` (global exception handler)

---

### traffic_stats.json

Location: `/data/traffic_stats.json`

Traffic counters. Updated every 30s.

```json
{
  "peers": {
    "u-u-i-d": {
      "today_total_bytes": 12345678,
      "week_total_bytes": 87654321,
      "today_total_human": "11.8 MB",
      "week_total_human": "83.6 MB"
    }
  },
  "global": {
    "today_total_bytes": 123456789,
    "week_total_bytes": 987654321,
    "today_total_human": "117.7 MB",
    "week_total_human": "941.9 MB"
  }
}
```

---

### traffic_stats.sqlite

Location: `/data/traffic_stats.sqlite`

Hourly and daily traffic history.

Schema:
```sql
CREATE TABLE traffic_hourly (
    peer_id TEXT,
    hour INTEGER,
    rx_bytes INTEGER DEFAULT 0,
    tx_bytes INTEGER DEFAULT 0,
    PRIMARY KEY (peer_id, hour)
);
CREATE TABLE traffic_daily (
    peer_id TEXT,
    day TEXT,
    rx_bytes INTEGER DEFAULT 0,
    tx_bytes INTEGER DEFAULT 0,
    PRIMARY KEY (peer_id, day)
);
```

---

### health_state.json

Location: `/data/health_state.json`

Transient health check cache. Regenerated every 30s. Loss is not critical.

---

### top_user_event.json

Location: `/data/top_user_event.json`

Last high-activity event for the dashboard.

---

## Backup Files

Location: `/data/backups/`

Files:
- `latest.wgadmin` — most recent backup
- `previous.wgadmin` — previous backup
- `YYYYmmdd_HHMMSS.wgadmin` — timestamped archives

Format: tar.gz.

Includes:
- clients.json
- settings.json
- speed_limits.json
- traffic_stats.json
- traffic_stats.sqlite
- activity.log
- health_state.json
- plus static web files

---

## Restore Safety Files

Location: `/data/restore-safety/`

Emergency snapshots created before restore operations.

---

## WireGuard Config

Location: `/data/wg0.conf`

Generated by entrypoint.sh from clients.json.
Do not edit manually — overwritten on every container start and peer sync.

---

## Git Ignore Policy

Must be excluded:
- `api_token`
- `*.json` runtime files
- `*.sqlite`
- `*.log`
- `*.wgadmin`
- `*.tar.gz`
- `venv/`
- `__pycache__/`

May be committed:
- `app.py`
- `web/*`
- `*.md`
- `Dockerfile`
- `entrypoint.sh`
- `install.sh`
- `requirements.txt`
- `init_config.py`
- `.gitignore`
