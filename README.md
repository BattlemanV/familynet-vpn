# Personal VPN Admin

Панель управления личным и семейным VPN-сервером на базе WireGuard / wg-easy.

## Стек

- Ubuntu 24.04
- Docker
- WireGuard
- wg-easy
- FastAPI
- PWA frontend

## Возможности

- управление клиентами
- QR-коды и конфиги
- статистика трафика
- журнал действий
- резервные копии
- восстановление
- ограничение скорости клиентов
